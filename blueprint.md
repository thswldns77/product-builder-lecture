
# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it uses modern web development practices.

## Project Outline

*   **HTML (`index.html`):** The main structure of the application.
*   **CSS (`style.css`):** The styles for the application, including a modern and bold design.
*   **JavaScript (`main.js`):** The logic for generating and displaying the lottery numbers.

## Current Task: Create Lottery Number Generator

*   **Objective:** Transform the initial "Hello World" page into a functional and visually appealing lottery number generator.
*   **Plan:**
    1.  **Modify `index.html`:** Update the title and body to include a container for the lottery numbers and a button to generate them.
    2.  **Modify `style.css`:** Add styles to create a visually engaging experience with a lottery machine theme.
    3.  **Modify `main.js`:** Implement the JavaScript logic to generate 6 unique random numbers from 1 to 45 and display them.
